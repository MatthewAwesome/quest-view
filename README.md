# quest-view
Oculus QuestにPCのデスクトップ画面を持ち込むアセットです。  
遅延は0.3秒程度です。音声も出力されます。  
webrtc_unity_pluginを使って実装しています。(参考文献：https://github.com/mhama/WebRtcUnityPluginSample)  

PC上で配信用のソフトウェアを実行する必要があります。

 # 配布ページ
